# Contributing to Anvil

Anvil is not open to code contributions at this time. The best ways to help are reporting bugs and suggesting features.

---

## Reporting Bugs

Open an issue and include:

- **Anvil version** and whether you are running from source or the pre-built binary
- **Windows version** (`winver`)
- **Target** — `--exe`, `--service`, or `--pid`, and whether it is a service
- **Module** that produced the unexpected result, if applicable
- **Full terminal output** — run with `--verbose` and paste the output
- **Expected vs actual behaviour**

---

## Requesting Features

Open an issue with the `enhancement` label. Describe:

- The attack class or scenario it covers
- Why existing modules don't address it
- Any relevant references (CVEs, research, tooling)
